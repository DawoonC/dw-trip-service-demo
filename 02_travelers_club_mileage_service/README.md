# 2. Traveler's Club Mileage Service

## Requirements
- Java >= 11
- Docker


## Run Application
```bash
# build
$ ./gradlew build

# start local server with docker
$ docker compose up
```
- 서버: http://localhost:5000
- DB 관리 페이지: http://localhost:5001
  - username: root
  - password: root
